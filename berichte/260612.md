### Theorieteil

wc ... word count
cut ... Spalten filtern
sed ... stream editor (suchen & ersetzten)
grep ... Zeile filtern


#### *word count: *
```sh
Eingabe:
wc klassenkassa.csv

Ausgabe:
 146  146 4384 klassenkassa.csv
 ```
 <!-- wc erklären -->
 1. 146 = Gibt die Zeilen aus.
 2. 146 = Anzahl der Wörter in der Datei.
 3. 4384 = Gibt die Anzahl der Bytes aus (Dateigröße).


#### _*cut:*_

```sh
Eingabe:

 cut -d, -f2 klassenkassa.csv | head

 Ausgabe:
 
 21.09.2023
 22.09.2023
 21.09.2023
 18.09.2023
 15.09.2023
 22.09.2023
 18.09.2023
 20.10.2023
 25.11.2023
 05.11.2023
 ```
Der cut Befehl schneidet bestimmte Spalten aus einer Datei heraus. Mit *-d* gibst du an, durch welches Ziel die einzelne Felder/Spalten getrennt sind. *-f2* gibt das 2. Feld aus.


#### *sed:*

```sh
Eingabe:

  echo "hello world" | sed "s/hello/hi/" 
  
  Ausgabe:
  
 hi world
 ```
 sed nimmt einen String als Eingabe und kann ihn verändern.
