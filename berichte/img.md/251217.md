# übung 3
 Übung 3.3 (Wildcards)
Was macht ls *.md oder ls *.txt? Recherchiere zu Wildcards in der shell, insbesondere *. Führe einige Experimente aus um die Bedeutung und Anwendung zu verstehen.

 ```sh
 eingabe: ls *.md

 ausgabe: README.md


 eingabe: ls *.txt

 Ausgabe:  etcdir.txt  hallo.txt  made_by_echoing.txt  out.txt  shopping.txt  test2.txt  test.txt
 ```
 erklärung:
 
 das zeichen * nennt man Wildcards(platzhalter).

 ls *.txt zeigt alle Textdateien an.

 ls *.md zeigt alle Markdown dateien an.
