## Aufgabe (nop)

Der nop-Befehl hat den Wert 0x90.

Der Sprungbefehl je besteht aus 6 Bytes.

*Der Vorgang:*

```sh
gdb ./pwcheck

b main

run

disas main

x/16xb 0x000055e3373751c8
```

*Danach wurden die 6 Bytes überschrieben:*

```sh

set {unsigned char}0x000055e3373751c8 = 0x90

set {unsigned char}0x000055e3373751c9 = 0x90

set {unsigned char}0x000055e3373751cA = 0x90

set {unsigned char}0x000055e3373751cB = 0x90

set {unsigned char}0x000055e3373751cC = 0x90

set {unsigned char}0x000055e3373751cD = 0x90
```


Ergebnis:


Passwort = G7pL9xQ2mR4tZcH

## Aufgabe (jne)

Hier wurde der je-Befehl verändert.

Er wurde durch jne ersetzt, dadurch wird die Bedingung umgedreht.

*Wert von jne: 0x85*

Schritte:

```sh
gdb ./pwcheck

b main

run

disas main

x/6xb 0x000055e3373751c8
```

Änderung:

```sh
set {unsigned char}0x000055e3373751c9 = 0x85
```
